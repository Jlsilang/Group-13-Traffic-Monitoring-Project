How to view traffic statistics for a road
1) Select the "Road" radio button.
2) Select the road you want to see from the "Select a main road" dropdown bar.
3) Select a time in 24 hour military format or select 0 to see general statistics.
4) Select weekend or weekday from the "Day Selection" dropdown bar.
5) Select a weather condition from the "Weather" dropdown bar.
6) Select "Show Live Traffic" radio button if you want to see live traffic conditions as well.
7) Select "View Statistics" button to view results.


How to view the traffic in a specific region
1) Select the radio button corresponding to the region you would like to view.
2) Select a time in 24 hour military format or select 0 to see general statistics.
3) Select weekend or weekday from the "Day Selection" dropdown bar.
4) Select a weather condition from the "Weather" dropdown bar.
5) Select "Show Live Traffic" radio button if you want to see live traffic conditions as well.
6) Select "View Statistics" button to view results.


How to reset the map
1) Select the "Reset" link at the bottom of the page or refresh the browser page.
